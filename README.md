# 常青女书 - Nushu Cultural Website

静态网页，展示女书文化，包含汉字转换、字符展览、传承人课程、每日一练四大模块。

## 技术栈
- 纯 HTML5 + CSS3 + Vanilla JavaScript
- 女书字体: Nushu.ttf（需放置于 字体/ 目录）
- 数据: nushu.txt（汉字-女书对照表）

## 页面结构
| 页面 | 文件 | 说明 |
|------|------|------|
| 首页 | index.html | 品牌介绍、四大功能入口 |
| 语言转换 | convert.html | 输入汉字转换为女书字符 |
| 字符展览 | gallery.html | 396个女书字符翻转展示 |
| 传承人课程 | courses.html | 免费视频 + 课程街付费入口 |
| 每日一练 | practice.html | 每日一句 + 临摹作品上传 |

## 使用方式
直接用浏览器打开 index.html 即可运行。

## 数据说明
- `数据/nushu.txt`: 汉字与女书字符的对照表（tab 分隔）
- `字体/Nushu.ttf`: 女书专用字体
- `网页图片/`: 装饰用海报图片
